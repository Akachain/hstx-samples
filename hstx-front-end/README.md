# Akachain - High Secure Transaction Samples ‚åÅ hstx-front-end

<h2>Quick Start</h2>

Go to directory
```js
cd hstx-front-end
```

Install dependencies
```js
npm install
```

Run in develop mode, application will be running at port 8080
```js
npm run serve
```

Build production for deployment
```js
npm run build
```
